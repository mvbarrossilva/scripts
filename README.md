
# Projeto de Análise Biológica de Peixes Teleósteos

Este projeto contém dados e scripts para análise biológica de amostras de peixes, incluindo:

- Importação e junção de dados de amostragem
- Análises morfométricas e estatísticas
- Visualizações por sexo, estágio de maturação e região

## Estrutura do projeto

- `data/`: arquivos CSV com os dados
- `scripts/`: script principal de análise em R (`analise_biologica.R`)

## Como usar

Abra o script em um ambiente R/RStudio e execute as células passo a passo.
